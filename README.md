1.  Python Flask-powered API with seven routes.

```  
@app.route("/")
@app.route("/age/")
@app.route("/age/<pclass>")
@app.route("/survived/")
@app.route("/survived/<pclass>")
@app.route("/gender-class/")
@app.route("/class-age/")
```
